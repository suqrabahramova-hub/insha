# Insha

Project management platform for engineers and architects with CAD/BIM file support.

## Features
- Project workspace and folder management
- CAD/BIM file upload and version control
- 2D/3D file viewer (IFC.js)
- Comments and markups
- User roles: Owner, Editor, Viewer
- Basic search and filter

## Tech Stack
- Frontend: React + TypeScript + Vite
- Backend: Node.js + NestJS
- Database: PostgreSQL
- Storage: AWS S3
- Viewer: IFC.js + Three.js
